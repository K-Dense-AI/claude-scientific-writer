## About This Fork

This repository is a fork of the Claude Scientific Writer plugin, currently undergoing customization. The original end-user documentation has been archived to `docs/*.original.*` for reference.

### Status

- **Active Development** - Plugin structure and skills are being modified
- **Not Production Ready** - Do not install as a Claude Code plugin
- **Breaking Changes Expected** - Original functionality may be altered or removed

### Original Documentation

The original upstream documentation is preserved in the `docs/` directory:

- `docs/README.original.md` - Original README with usage instructions
- `docs/CLAUDE.original.md` - Original CLAUDE.md agent instructions
- `docs/CHANGELOG.original.md` - Upstream version history
